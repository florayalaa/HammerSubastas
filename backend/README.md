# Backend para TPO_DA2

Este directorio contiene la API del backend para el proyecto móvil.

## Estructura

- `src/index.ts`: punto de entrada.
- `src/app.ts`: configuración de Express y rutas.
- `src/routes/`: controladores de rutas.

## Comandos

- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`

## APIs iniciales

- `GET /health`
- `POST /auth/login`
- `POST /auth/register`
- `GET /auctions`
- `GET /auctions/:id`
- `GET /users/profile`
